# Dewy website

## Development

* Get nodemon

		npm install nodemon -g
		
* Use it to restart the app upon changes

		nodemon app.js


## Installation

* Install dependencies:

		npm install
		
* Run node:

		node app.js
		

## Usage

* Go to localhost:3000 to visit the site
