# Dewy website

## Installation

* Configure OAuth client and JWT encryption:

        cp config.json.default config.json

* Install dependencies:

        npm install
        
* Run node:

        npm start

## Development

* Run bower to download packages:

        bower install

* Run grunt to compile assets

        grunt

## Usage

* Go to localhost:3000 to visit the site
