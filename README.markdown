# Dewy website

## Installation

* Configure OAuth client secret and JWT encryption:

        cp config.json.default config.json

* Install dependencies:

        npm install

## Development

* Run bower to download packages:

        bower install

* Run grunt to compile assets

        grunt

## Usage

* Run node:

        npm start

* Go to localhost:3000 to visit the site
