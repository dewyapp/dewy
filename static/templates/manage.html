Manage sites

Add a site

To add a site to Dewy, you must first download and enable the Dewy module on any site you wish to add. Download the module from Drupal.org.

Your API Key

Your API key is what links sites to your account. Keep it confidential. You can reset your API key to clear all sites' ability to associate with your account.
